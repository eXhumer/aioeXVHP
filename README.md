# aioeXVHP
Asynchronous Python library for managing videos hosted from various supported video hosting platforms.

## Supported Platforms
* JustStreamLive
* Mixture
* Streamable
* Streamff
* Streamja

## Package Installation Guide
### Requires
* Python 3.9+
* pip package manager

### Install from source
```console
pip install .
```
